# Wifi QR-Code

Creation of a QR Code for connecting your device to the Wifi.

You have to enter the password of your wifi network, the ssid (the name of the network) and also the authentification type (that can be "WPA", "WPE", "WPA2" and "WPA3" but the most common is "WPA").

You can also change the name of the final picture with the QR Code, by default it is "wifi_qr_code.png".

There is an example in the repository of a QR code that return the script.
